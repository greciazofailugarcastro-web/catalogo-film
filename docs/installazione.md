# Installazione

## Come aprire il progetto

### Opzione 1 - Sito online
Apri direttamente nel browser:
https://greciazofailugarcastro-web.github.io/catalogo-film/

### Opzione 2 - In locale
1. Scarica o clona la repository:
   git clone https://github.com/greciazofailugarcastro-web/catalogo-film.git

2. Apri la cartella:
   cd catalogo-film

3. Apri index.html nel browser

## Requisiti
- Browser moderno
- Nessuna dipendenza richiesta
