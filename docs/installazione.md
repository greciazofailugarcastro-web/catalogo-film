# Installazione

## Requisiti

- Python 3.x
- Git installato

## Clonare la repository

```bash
git clone https://github.com/greciazofailugarcastro-web/grecialu.git
cd progetto_git
```

## Installare le dipendenze

```bash
pip install -r requirements.txt
```

## Avviare il progetto

```bash
python main.py
```

## Clonare la repository

Aprire il terminale ed eseguire:

```bash
git clone https://github.com/greciazofailugarcastro-web/grecialu.git
cd progetto_git
```
