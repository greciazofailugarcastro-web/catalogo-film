# Quick Start

## 1. Scarica il progetto

git clone https://github.com/greciazofailugarcastro-web/grecialu.git

## 2. Apri la cartella

cd progetto_git

## 3. Installa le dipendenze

pip install -r requirements.txt

## 4. Avvia il progetto

python main.py

## 5. Pronto!

Il progetto è in esecuzione.
