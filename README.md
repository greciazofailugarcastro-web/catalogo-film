# Catalogo Film 🎬

Sito statico per la consultazione di film.

## Obiettivo

Progetto realizzato per il corso FISM 2026.
Sito statico in HTML e CSS con documentazione completa.

## Funzionalità

- Ricerca per titolo
- Filtro per genere
- Visualizzazione dettagli film
- Tabella comandi Git
- Modulo di contatto
