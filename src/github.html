Comandi principali

Comando Descrizione
git clone Clona una repository
git status Mostra lo stato della repository
git add Aggiunge modifiche allo staging
git commit Salva le modifiche
git push Carica le modifiche su GitHub
git pull Scarica le modifiche da GitHub
